Síntese

O curso de Desenvolvimento Web abrange a criação e estilização de páginas web utilizando HTML e CSS. HTML é usado para estruturar o conteúdo das páginas, enquanto o CSS é responsável pela aparência visual. Juntas, essas tecnologias permitem a construção de sites modernos, funcionais e visualmente atraentes.

HTML

HTML (Linguagem de Marcação de Hipertexto) é a base da web. Ele define a estrutura e o conteúdo das páginas, utilizando elementos como títulos, parágrafos, imagens e links. Durante o curso, foram criadas várias páginas web que demonstram a aplicação prática de HTML para organizar e apresentar informações de maneira clara e eficiente.

CSS

CSS (Folhas de Estilo em Cascata) é utilizado para estilizar e melhorar a apresentação das páginas HTML. Com CSS, é possível alterar cores, fontes, layouts e adicionar animações. No curso, foram realizados projetos que aplicam CSS para criar designs responsivos e atraentes, melhorando a experiência do usuário em diferentes dispositivos.
